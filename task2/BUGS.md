Баг 1: Не получается создать объявление
Эндпоинт: POST /api/1/item
Шаги:
Отправить POST запрос с данными:
json
123456
Ожидаемый результат: Статус 200, объявление создано
Фактический результат: Статус 400 Bad Request
Важность: Высокая (не работает основная функция)

Баг 2: Возвращает все объявления вместо нужных
Эндпоинт: GET /api/1/:sellerID/item
Шаги:
Отправить GET запрос на /api/1/999999/item (несуществующий продавец)
Ожидаемый результат: Пустой массив [] или ошибка 404
Фактический результат: Возвращается 1250 объявлений всех продавцов
Важность: Критическая (проблема безопасности)
-----
Баг 3: Неправильный статус для несуществующего объявления
