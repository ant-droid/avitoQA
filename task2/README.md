**Инструкция по запуску тестов**
- **Описание**: Набор простых pytest-тестов для API микросервиса Avito. Тесты находятся в файле `tests.py` и делают реальные HTTP-запросы к `https://qa-internship.avito.com`.

- **Требования**: установленный Python 3.8+ и доступ в интернет

- **Рекомендуемые зависимости**: `requests`, `pytest` (и дополнительный плагин генерирования HTML-отчёта `pytest-html`, опционально).

**Шаги — подготовка окружения**:

- Откройте PowerShell и переключитесь в папку с тестами:

```
cd C:\Study\Tests\task2
```

- Создайте (опционально) виртуальное окружение и активируйте его:

```
python -m venv .venv; .\.venv\Scripts\Activate.ps1
```

- Установите зависимости вручную:

```
pip install requests pytest pytest-html
```

- Если вы хотите использовать `requirements.txt`, добавьте в него нужные пакеты и выполните:

```
pip install -r .\requirements.txt
```

**Шаги — запуск тестов**:

- Запуск всех тестов (компактный вывод):

```
pytest -q
```

- Запуск с подробным выводом:

```
pytest -v
```

- Запуск и генерация HTML-отчёта (если установлен `pytest-html`):

```
pytest -v --html=report.html
```

После выполнения файл `report.html` появится в текущей папке.
